Wireless Hacking

 

 

A wireless network connects two or more devices using radio waves in a confined area. Wireless network devices may move while staying connected and sharing data. Their installation is inexpensive and rapid, which is why they're so popular.

Wireless networks are popular and simple to install. IEEE 802.11 is used. Wireless routers link users to the Internet in wireless networks.

Access Points extend wireless ranges and act as logical switches in wireless networks. Wireless networks are flexible yet insecure. A hacker can sniff network traffic remotely. Hackers may easily sniff wireless networks using radio waves.

Most attackers employ network sniffing to identify a wireless network's SSID. In monitor mode, our wireless cards sniff.

Kismet:is a Kali distribution wireless sniffer tool. It's also available at https://www.kismetwireless.net.

NetStumbler:NetStumbler is a Windows-only wireless hacking tool. http://www.stumbler.net/ has it. NetStumbler is straightforward to use.

Wired Equivalent Privacy:Wired Equivalent Privacy (WEP) secures and protects wireless networks. Data connection encryption prevents unwanted network access. Before transmission, the key encrypts packets. A packet integrity checker checks for changes after transmission.

Some others

  • WEPcrack
  • Aircrack-ng

 

 

Wireless DoS Attacks:An attacker may assault a wireless network from a distance, making it harder to gather proof. The first form of DoS is Physical Attack. This form of assault is simple and may be caused via 2.4 GHz cordless phones.

Network DoS Attack:Network DoS Attack is another. As the Wireless Access Point provides a shared medium, it allows traffic to overwhelm the AP, slowing its processing for clients trying to connect. Ping flood DoS attacks may cause such assaults.

LOIC is another DoS tool.

 

 

Course Curriculum

Lab: COWPATTY

Find the COWPATTY utility

This tool, which is used to hack the most secure security systems, is included by default in the Kali Linux software package. It is also interesting to know that this tool is located in the /usr/local/bin directory, so it can be launched from any folder in Kali Linux. To find it in Kali, run the following code:

cd /usr/local/bin

Use the COWPATTY help page

Just type cowpatty to find this tool. Then, Kali Linux will show you a small help page. Note that coWPAtty needs some data to work properly, including the following:

  • Word list
  • File with password hash
  • Access point SSID for hacking

Switch the wireless adapter to monitor mode

Like other hacking tools, coWPAtty needs to change the Wi-Fi adapter to monitor mode to get the password, for this you can use the following code:

airmon-ng start wlan0

Create a trace file

In this step, you need to create a file to store the intercepted password. For this purpose, use the following code:

airodump-ng â€“bssid 00: 25: 9C: 97: 4F: 48 -c 9 -w cowpatty mon0

The above command dumps the selected access point (00:25:9C:97:4F:48), the specified channel (-c 9) and saves the hash to a file called cowcrack.

Wait

Now you just have to wait for someone to connect to the access point you modified, where you can intercept the hash. Use Airodump-ng to get notified when someone connects to the target network.

Run the coWPAtty tool

After receiving the hashed password, we can use coWPAtty and decrypt the desired hash using our own word list. To do this, run the following code:

cowpatty -f /pentest/passwords/wordlists/darkc0de.lst -r /root/cowcrack-01.cap -s Mandela2

coWPAtty creates a hash with the SSID for each word in the list and compares it to the intercepted hash. After the hash is matched, the coWPAtty tool will display the password of the access point on the screen.

Generate your hash

Normally, coWPAtty uses a list of words provided by the user and calculates the hash of each one based on the specified SSID, although this consumes a lot of CPU resources. Fortunately, the tool now supports working with a ready-to-use hash file, which makes cracking WPA2-PSK passwords even more efficient. The following code can help you:

genpmk -f /pentest/passwords/wordlists/darkc0de.lst -d hashes -s Mandela2

Use your hash

After creating a list of hashes for a particular SSID, we can start cracking the password by running the following command:

cowpatty -d hashfile -r dumpfile -s ssid

Using this tool, you can easily hack wpa2 psk wifi without complications.
